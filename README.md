# 📖 About the Repository

**Python Mini Projects** is a collection of beginner-friendly applications built while learning Python programming. Each project focuses on applying core Python concepts to create interactive command-line programs using clean, organized, and easy-to-understand code.

The repository emphasizes practical learning by transforming theoretical concepts into working applications. It is continuously updated with new projects as I progress through my Python learning journey.

# 📚 Projects

| Project | Description | Concepts Covered |
|----------|-------------|------------------|
| 🎮 Rock Paper Scissors | Classic game where the user plays against the computer. | Random Module, Loops, Conditional Statements |
